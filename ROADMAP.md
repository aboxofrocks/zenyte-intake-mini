# Zenyte Document Management - Feature Roadmap

## Completed Features
- Document Hub UI with tabs (Financial, Corporate, Compliance, Contracts, Other DD)
- DD Readiness score bar with sparkline trends
- Financial document extraction with Claude API
- Checklist visualization with Apply Template
- Phase 3 - Unified Documents view
- Link documents to checklist items
- Expiration tracking for documents
- Document versioning
- Auto-categorization (filename patterns)
- Bulk upload with auto-sort
- Document notes with type guidance
- DD Readiness trends over time
- Missing docs report across all practices
- Dashboard overview (readiness scores, expiring docs, missing items)
- Email/Slack notifications (notification center, Slack webhooks, email digest)
- Global document search (Ctrl+K, searches docs by name/type/notes)
- Export to CSV/PDF (practices, documents, DD readiness report)
- Document request workflow (request -> received -> reviewed tracking)
